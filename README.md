# Prompt Tests

A repository for experimenting with, testing, and refining prompts for Large Language Models. Each project captures a complete prompt-driven development lifecycle—from specification through implementation to documented results.

---

## Repository Structure

```
prompt-tests/
├── README.md
├── cv-printer/
│   ├── printer-cv.html
│   ├── README.md
│   ├── PROMPT.MD
│   ├── demo/
│   ├── chat-log/
│   └── misc/
```

Each project directory contains:
- `PROMPT.MD` — the original prompt specification
- Deliverable files (code, assets, etc.)
- `README.md` — project documentation and results
- `chat-log/` — conversation history and iteration notes (optional)
- `demo/` — visual demonstrations (optional)
