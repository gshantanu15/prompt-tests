Printer CV

Scroll-to-Print Animated CV · Single HTML File

A single-page animated printer simulation built entirely with HTML, CSS, and vanilla JavaScript. Scrolling feeds a sheet of paper out of a printer, revealing a CV line-by-line — just like watching a real printer at work. Created as a prompt-driven development test using GitHub Copilot.

HTML CSS JavaScript Prompt Engineering GitHub Copilot
  • Paper feeds bottom-up matching real printer physics
  • Print head with animated blue sweep at printing edge
  • Status LED pulses amber during active printing
  • Initial prompt produced ~550 lines in one shot
  • Physics correction required only 3 CSS + 2 JS edits
  • Zero dependencies — pure HTML, CSS, JS

How Each Project Is Built

📝

Specify

Write a detailed natural-language prompt describing the desired outcome, constraints, and interaction model.

⚡

Generate

Feed the prompt to an LLM and receive a working first draft — often hundreds of lines in one shot.

🔧

Refine

Iterate with targeted follow-up prompts. Small, focused corrections often beat full rewrites.

📚

Document

Capture the prompt, chat log, demo assets, and lessons learned for reproducibility.
